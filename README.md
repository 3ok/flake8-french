flake8-french
============

A flake8 plugin that enforces raw f-strings to be fr"foo" instead of rf"foo"

## installation

`pip install flake8-french`

## flake8 codes

| Code   | Description                                                    |
|--------|----------------------------------------------------------------|
| FRE001 | 1:4: FRE001 Use 'fr' to specify a raw f-string instead of 'rf' |
